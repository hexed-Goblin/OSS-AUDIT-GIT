## Project Briefing

This project is about Git. Git is a version control system that a lot of people use. We are going to look at where Git came from and what kind of license it has. We will also talk about the ethics of using Git and what other tools work with Git. Then we will show how to use Git on a Linux computer.
The project also includes writing scripts for the computer. These scripts will show how to do things automatically using Bash. We want to see how Bash can help us with system operations and automation. The project is really, about Git and how to use Bash with Git.

## Scripts Included

### 1. System Identity Report (`script1.sh`)

Displays system details such as OS name, kernel version, current user, uptime, date, and license information.

### 2. Git Package Inspector (`script2.sh`)

Checks whether Git is installed in the system and displays its version along with a message about open source philosophy.

### 3. Disk and Permission Auditor (`script3.sh`)

Analyzes system directories and displays their permissions and disk usage.

### 4. Log File Analyzer (`script4.sh`)

Processes a log file, counts the number of error messages, and displays the last few error entries.

### 5. Manifest Generator (`script5.sh`)

Takes user input and generates a personalized manifesto saved as a text file.
